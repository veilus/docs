title: Keyboard Shortcuts
description: Essential keyboard shortcuts for Veilus.
---

## General

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + N` | New profile |
| `Ctrl/Cmd + Shift + N` | New incognito profile |
| `Ctrl/Cmd + W` | Close current tab |
| `Ctrl/Cmd + T` | New tab |
| `Ctrl/Cmd + L` | Focus address bar |
| `Ctrl/Cmd + Shift + T` | Reopen closed tab |
| `F11` | Toggle fullscreen |
| `F5` | Reload page |
| `Ctrl/Cmd + Shift + I` | Open DevTools |

## Profile Manager

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Shift + P` | Open profile manager |
| `Enter` | Launch selected profile |
| `Delete` | Delete selected profile |
| `Ctrl/Cmd + D` | Duplicate profile |
| `Ctrl/Cmd + E` | Edit profile settings |

## VeilusFlow

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Shift + R` | Start/stop recording |
| `Ctrl/Cmd + Shift + F` | Open VeilusFlow panel |
| `Ctrl/Cmd + Enter` | Run current flow |
| `Escape` | Stop running flow |
